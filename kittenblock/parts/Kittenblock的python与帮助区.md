# 舞台python与帮助区

## 切换到舞台Python模式

python按钮是用来将图形块工作区切换到python编程工作区。python的编程对象就是舞台的小喵。 

- 小喵独创的拖拽积木块转换python代码   
- 通过拖拽可以快速提醒对应操作的python控制指令。
